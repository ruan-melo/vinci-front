A social network that allows users share photos
## Features

- Push Notification
- Responsive
- Image upload

## Tech Stack

**Client:** Next.js and TailwindCSS

**Server:** NestJS, GraphQL and Firebase (Realtime Database, Cloud Messaging) [API repository](https://github.com/ruan-melo/vinci-back)
